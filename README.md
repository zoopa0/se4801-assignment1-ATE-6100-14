# Project Description

ShopWave Starter is a Spring Boot 3.x application for managing products in a ShopWave system. It demonstrates basic CRUD operations, service-layer logic, and uses H2 database for testing.

# Features
Add, view and update products
Basic validation and error handling
Service layer implementation for business logic
H2 in-memory database for testing purposes
Exposes /actuator endpoint for monitoring
How to Run

Clone the repository:

git clone https://github.com/zoopa0/se4801-assignment1-ATE-6100-14

Navigate to the project folder:

cd shopwave-starter

Run the application:

mvn spring-boot:run
The application starts on http://localhost:8080
How to Test

Run unit tests with:

mvn test


# Technologies
Java 21
Spring Boot 3.x
H2 Database (in-memory)
Spring Data JPA
Lombok
Spring Boot Actuator
